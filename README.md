# MobileProgrammingFinal
Mobile Programming Final Exam Project Case (Take Home Project - 1 Week)

Description:

Mobile Apps that provides real-time up-to-date news (from News API) and user can search news based on keywords. 
User also can save the news locally such that if the mobile data/Wi-Fi connection is unavailable, user still can read the saved news.
